# Write quizzes in Markdown. Run them in the terminal.

## What is QuizMD?

QuizMD is a terminal-first quiz tool for classes, revision, workshops, and local assessments.

It helps you go from a simple `.md` file to a playable quiz fast, while supporting deeper learning modes when you need them.

## Run in 60 seconds

```bash
pip install quizmd
quizmd init
quizmd --validate hello-quiz.md
quizmd hello-quiz.md
```

## Why people use it

- Teaching: run lightweight quizzes in classrooms or labs.
- Revision: reinforce concepts with fast repeated practice.
- Workshops: ship starter files and keep everyone on one flow.
- Local assessments: run offline-friendly quizzes in terminals.

When users see these use cases immediately, retention improves because product shape is obvious from the first screen.
