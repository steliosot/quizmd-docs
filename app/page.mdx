# QuizMD Documentation

QuizMD is a terminal-first quiz application for MCQ, imposter, essay, and room-based multiplayer modes.

## What you can do

- Run local markdown quizzes in your terminal.
- Use imposter mode to identify misleading answer options.
- Grade essay responses with AI providers.
- Create online room sessions for compete, collaborate, and boxing modes.
