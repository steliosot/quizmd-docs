# Create Your First Quiz

## Anatomy of a quiz file

A basic MCQ file has:

- Quiz title: `# ...`
- Question block: `## ...`
- Options: lines starting with `-`
- Required metadata: `Answer:`, `Type:`
- Optional metadata: `Time:`, `Explanation:`, `Imposters:`

## Complete small file

```md
# Intro Quiz

## Question 1
What is 2 + 2?

- 3
- 4
- 5

Answer: 2
Type: single
Time: 20
Explanation: 2 + 2 equals 4.
```

## Validate and run

```bash
quizmd --validate intro-quiz.md
quizmd intro-quiz.md
```
