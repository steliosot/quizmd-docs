# Create Your First Quiz

QuizMD quizzes are plain Markdown templates. Start from one working file, change the text, validate it, then run it.

```bash
quizmd init
```

`quizmd init` creates starter templates for Classic, Imposter, Debug, and Essay modes. Room modes reuse a normal quiz template and run it online.

Classic, Imposter, and Debug templates run locally with no API key. Essay templates need an AI provider key. Room templates need internet access because they connect to an online room server.

## First template

Use this as the smallest Classic Mode quiz:

```md
# Intro Quiz

## Question 1
What is 2 + 2?

- 3
- 4
- 5

Answer: 2
Type: single
Time: 20
Explanation: 2 + 2 equals 4.
```

```bash
quizmd --validate intro-quiz.md
quizmd intro-quiz.md
```

## Anatomy

- `# Intro Quiz`: quiz title.
- `## Question 1`: question block.
- `- 3`: answer option.
- `Answer: 2`: correct option number.
- `Type: single`: question type.
- `Time: 20`: optional timer.
- `Explanation: ...`: optional feedback after answering.

Use Classic Mode for normal single-choice or multiple-choice practice. It is the best first template because it works locally and needs no AI key.

```md
# Classic Quiz

## Question 1
Which are built-in Python types?

- list
- banana
- dict
- int

Answer: 1,3,4
Type: multiple
Time: 25
Explanation: `list`, `dict`, and `int` are built-in Python types.
```

```bash
quizmd --validate classic-quiz.md
quizmd classic-quiz.md
```

Use Imposter Mode when you want learners to spot misleading wrong options, not just pick the correct answer.

An imposter is a wrong option that sounds plausible. Example: if the correct answer is “append adds one item”, then “append adds each list element separately” is an imposter because it describes `extend`, not `append`.

```md
# Imposter Quiz

## Question 1
What does `arr.append([4,5])` do?

- Adds the list `[4,5]` as one element
- Adds `4` and `5` as separate elements
- Replaces the last element with `[4,5]`
- Extends the list with `[4,5]`

Answer: 1
Imposters: 2,4
Type: single
Time: 30
Explanation: append adds one element. extend adds the contents separately.
```

```bash
quizmd --validate imposter-quiz.md
quizmd imposter-quiz.md
```

Use Debug Mode when you want students to repair broken code and learn by fixing syntax, indentation, and logic mistakes.

````md
# Debug Quiz: Python Foundations

## Question 1
There are two errors. Fix the function so it runs correctly.

Broken:
```python
def greet(name)
    print("Hello, " + name
```

Fixed:
```python
def greet(name):
    print("Hello, " + name)
```

Type: debug
Hint: Start at line 1, then check missing punctuation on line 2.
Explanation: Function definitions need a trailing colon, and print call needs a closing parenthesis.
````

```bash
quizmd --validate debug-quiz.md
quizmd debug-quiz.md
```

Use Essay Mode for short written answers, rubric scoring, and feedback. Essay templates use dedicated sections and need an AI provider key before grading.

```md
# Essay Question: requirements basics

## Question
What is `requirements.txt` and why is it useful in Python projects?

## Instructions for Students
Write 5-10 lines.

## Evaluation Criteria (Total: 3 points)
1. **Dependency listing (1 point)**
- Mentions project packages are listed in one file
2. **Reproducibility (1 point)**
- Mentions same environment across machines
3. **Collaboration (1 point)**
- Mentions easier teamwork

## Reference Answer
`requirements.txt` lists dependencies so everyone can install the same environment.

## AI Evaluation Rules
Evaluate only by the rubric above.

## Output Format
Score: XX%

Feedback:
- What worked
- What is missing
- One suggestion
```

```bash
export GEMINI_API_KEY="your_key_here"
quizmd --validate requirements-essay.md
quizmd requirements-essay.md
```

Windows PowerShell:

```powershell
$env:GEMINI_API_KEY="your_key_here"
quizmd requirements-essay.md
```

Use Room: Compete when you want a fast group quiz where speed and correctness both matter. The quiz file is a normal Classic template.

```md
# Speed Round

## Question 1
What command creates starter QuizMD files?

- quizmd init
- quizmd start
- quizmd new

Answer: 1
Type: single
Time: 20
Explanation: `quizmd init` creates starter templates.
```

```bash
quizmd room --create --mode compete --quiz speed-round.md
# share the printed room name and token
quizmd room --join <room-name> --token <room-token>
```

Use Room: Boxing for a teacher/student conversation. The quiz file gives the room context, then chat and follow-up questions carry the session.

```md
# Boxing Session

## Question 1
Explain what a Python virtual environment is.

- It isolates project dependencies
- It deletes old packages
- It replaces Python

Answer: 1
Type: single
Time: 30
Explanation: A virtual environment keeps project dependencies separate.
```

```bash
quizmd room --create --mode boxing --role teacher --quiz boxing-session.md
# share the printed room name and token
quizmd room --join <room-name> --token <room-token> --role student
```
