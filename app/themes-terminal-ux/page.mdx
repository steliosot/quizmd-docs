# Themes and Terminal UX

## Theme options

```bash
quizmd --theme auto file.md
quizmd --theme light file.md
quizmd --theme dark file.md
```

## Light theme

Use the light theme when the terminal is projected, shared on a bright screen, or used in a classroom with strong lighting.

```bash
quizmd --theme light hello-quiz.md
```

![QuizMD light terminal theme example](/theme-light-terminal.svg)

## Dark theme

Use the dark theme for low-light rooms, long solo sessions, or terminals already configured with a dark background.

```bash
quizmd --theme dark hello-quiz.md
```

![QuizMD dark terminal theme example](/theme-dark-terminal.svg)

## When to use each

- `auto`: default for mixed environments.
- `light`: bright terminals and projected classrooms.
- `dark`: low-light or long sessions.

## Readability tips

- For projectors, prefer `--theme light` and a larger terminal font.
- In labs, keep contrast high and avoid busy terminal backgrounds.
- Use `--no-color` when terminal rendering is inconsistent.
