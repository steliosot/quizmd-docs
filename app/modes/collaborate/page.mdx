# Room: Collaborate

Collaborate is the team mode. The group works toward one fully correct outcome per question instead of competing individually.

![Room Collaborate terminal example using hello-quiz.md](/mode-collaborate-terminal.svg)

## Start a room

```bash
quizmd room --create --mode collaborate --quiz debugging-quiz.md
```

Players join with the shared room name:

```bash
quizmd room --join <room-name>
```

## Rules

- Work as a team to reach one fully correct outcome per question.
- Everyone submits; if consensus is wrong, the team retries.
- Team score increases only when all required answers are correct.
- Finish all questions with the highest team score possible.
- Press `Ctrl+C` to exit at any time.

## Best for

- Concept checks after teaching.
- Group discussion.
- Teams learning how to justify answers.

## Example quiz moment

```md
## Question 1
Which changes make this API client more reliable?

- Add a timeout to requests
- Retry every failed request forever
- Handle pagination
- Log the API key for debugging

Answer: 1,3
Type: multiple
Time: 90
Explanation: Timeouts and pagination improve reliability. Infinite retries and logging secrets create new problems.
```
