# Room: Collaborate

Collaborate is the team mode. The group works toward one fully correct answer per question instead of competing individually.

![Room Collaborate terminal example using hello-quiz.md](/mode-collaborate-terminal.svg)

Room modes require internet access to reach the room server. They do not need an AI API key.

## Start a room

```bash
quizmd room --create --mode collaborate --quiz hello-quiz.md
```

Players join with the room name and token printed by the host:

```bash
quizmd room --join <room-name> --token <room-token>
```

## Rules

- Work as a team to reach one fully correct answer per question.
- Everyone submits; if consensus is wrong, the team retries.
- Team score increases only when all required answers are correct.
- Finish all questions with the highest team score possible.
- Use `/quit` to leave the room cleanly. `Ctrl+C` exits local quiz mode.

## Best for

- Concept checks after teaching.
- Group discussion.
- Teams learning how to justify answers.

## Example quiz moment

```md
## Question 1
Which changes make this API client more reliable?

- Add a timeout to requests
- Retry every failed request forever
- Handle pagination
- Log the API key for debugging

Answer: 1,3
Type: multiple
Time: 90
Explanation: Timeouts and pagination improve reliability. Infinite retries and logging secrets create new problems.
```
