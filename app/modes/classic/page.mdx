# Classic Mode

Classic Mode is the standard MCQ flow for single and multiple choice.

![Classic Mode terminal example from hello-quiz.md](/mode-classic-terminal.svg)

## Behavior

- Supports `Type: single` and `Type: multiple`.
- Uses optional `Time:` limits.
- Shows explanation after each question when present.
- Requires no AI configuration.

## Tiny example

```md
# Python File Handling

## Question 1
Which pattern closes a file automatically after reading it?

- f = open("data.txt")
- with open("data.txt") as f:
- read("data.txt")
- close("data.txt")

Answer: 2
Type: single
Time: 30
Explanation: A `with` block uses a context manager, so the file is closed even if an error happens.
```
