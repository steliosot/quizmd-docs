# Classic Mode

Classic Mode is the standard MCQ flow for single and multiple choice.

## Behavior

- Supports `Type: single` and `Type: multiple`.
- Uses optional `Time:` limits.
- Shows explanation after each question when present.
- Requires no AI configuration.

## Tiny example

```md
# Python Basics

## Question 1
What is 2 + 2?

- 3
- 4
- 5

Answer: 2
Type: single
Time: 20
Explanation: 2 + 2 equals 4.
```

```bash
quizmd --validate my-quiz.md
quizmd my-quiz.md
```
