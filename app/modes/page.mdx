# Modes

QuizMD has three core learning experiences plus online room variants.

## A) Classic Quiz Mode

- Single-choice and multiple-choice questions.
- Optional timers per question.
- Explanation output after submission.
- No AI key required.

[Open Classic Mode details](/modes/classic)

## B) Imposter Mode

- Same quiz flow as Classic mode.
- Users also identify misleading options.
- Great for misconception spotting and deeper understanding.

[Open Imposter Mode details](/modes/imposter)

## C) Essay Mode

- Markdown-defined essay questions and rubric sections.
- Rubric-based evaluation with optional AI providers.
- Distinct structure from MCQ files and separate runtime settings.

[Open Essay Mode details](/modes/essay)

## Room modes (online)

- `compete`: speed-based scoring.
- `collaborate`: consensus required.
- `boxing`: teacher/student chat + scoring.

```bash
quizmd room --create --mode compete --quiz hello-quiz.md
quizmd room --create --mode collaborate --quiz hello-quiz.md
quizmd room --create --mode boxing --quiz hello-quiz.md
quizmd room --join <room-name>
```
