# Modes

QuizMD has three core learning experiences plus online room variants. Start with Classic Mode, then use Imposter or Essay Mode when you want deeper learning.

## A) Classic Quiz Mode

- Single-choice and multiple-choice questions.
- Optional timers per question.
- Explanation output after submission.
- No AI key required.

```md
## Question 1
What is 2 + 2?

- 3
- 4
- 5

Answer: 2
Type: single
Time: 20
Explanation: 2 + 2 equals 4.
```

[Open Classic Mode details](/modes/classic)

## B) Imposter Mode

- Same quiz flow as Classic mode.
- Users also identify misleading options.
- Great for misconception spotting and deeper understanding.

```md
Answer: 1
Imposters: 4
Type: single
```

[Open Imposter Mode details](/modes/imposter)

## C) Essay Mode

- Markdown-defined essay questions and rubric sections.
- Rubric-based evaluation with optional AI providers.
- Distinct structure from MCQ files and separate runtime settings.
- Produces score and feedback sections for review.

[Open Essay Mode details](/modes/essay)

## Room modes (online)

Room modes let several people use the same quiz content together.

- [Room: Compete](/modes/compete): answer correctly as fast as possible.
- [Room: Collaborate](/modes/collaborate): work as a team until the answer is fully correct.
- [Room: Boxing](/modes/boxing): teacher/student chat, follow-ups, scoring, and transcript saving.

```bash
quizmd room --create --mode compete --quiz hello-quiz.md
quizmd room --create --mode collaborate --quiz hello-quiz.md
quizmd room --create --mode boxing --quiz hello-quiz.md
quizmd room --join <room-name>
```
