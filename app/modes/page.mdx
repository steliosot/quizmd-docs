# Modes

QuizMD has six learning modes: three file-based quiz modes and three live room modes. Use the same Markdown-first workflow, then choose the mode that fits the session: solo practice, misconception spotting, rubric feedback, racing, teamwork, or live oral assessment.

## Classic Mode

Classic Mode is the clean starting point: ask a question, show options, collect one or more answers, then show feedback.

```md
## Question 1
Which Python data type removes duplicate values?

- list
- set
- tuple
- dict

Answer: 2
Type: single
Explanation: A set keeps unique values, so duplicates are removed.
```

## Imposter Mode

Imposter Mode adds a second thinking step. Learners still choose the correct answer, but they also learn to spot the misleading option that sounds plausible.

```md
## Question 1
Which command creates a Python virtual environment?

- python -m venv .venv
- pip install venv
- source .venv/bin/activate
- python requirements.txt

Answer: 1
Imposters: 2
Type: single
Explanation: `pip install venv` sounds plausible, but `venv` is run as a Python module.
```

## Essay Mode

Essay Mode is for written answers and rubric feedback. It uses a different file structure from MCQ quizzes and can evaluate with Gemini, OpenAI, or Anthropic.

```md
# Essay Question: Explain dependency pinning

## Question
Explain why a Python project might pin dependencies in `requirements.txt`.

## Evaluation Criteria
- Mentions reproducible installs
- Explains team or CI consistency
- Notes the tradeoff with updates/security
```

## Room: Compete

Compete turns a normal quiz into a live race. Use it when you want energy, quick recall, and a visible final score.

```bash
quizmd room --create --mode compete --quiz python-basics.md
quizmd room --join <room-name>
```

## Room: Collaborate

Collaborate turns the same quiz content into a team challenge. It is useful when the answer needs discussion, like debugging, API design, or code review.

```bash
quizmd room --create --mode collaborate --quiz debugging-quiz.md
quizmd room --join <room-name>
```

## Room: Boxing

Boxing is a teacher/student room for live questions, natural follow-ups, scoring, and a saved transcript.

```bash
quizmd room --create --mode boxing --role teacher --quiz code-review.md
quizmd room --join <room-name> --role student
```
