# Imposter Mode

Imposter Mode extends MCQ with an extra task: identify misleading wrong options.

![Imposter Mode terminal example from hello-imposter.md](/mode-imposter-terminal.svg)

Imposter Mode works locally and needs no API key.

## Why use it

- Strengthens conceptual understanding.
- Surfaces common misconceptions.
- Improves learning depth vs memorization-only answering.

## Example pattern

```md
## Question 1
Which command creates a Python virtual environment?

- python -m venv .venv
- pip install venv
- source .venv/bin/activate
- python requirements.txt

Answer: 1
Imposters: 2
Type: single
Time: 30
Explanation: `pip install venv` looks believable, but `venv` is built into Python and runs with `python -m venv`.
```

- Correct answer: `1`
- Imposter answer(s): `2`

```bash
quizmd --validate hello-imposter.md
quizmd hello-imposter.md
```
