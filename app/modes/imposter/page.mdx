# Imposter Mode

Imposter Mode extends MCQ with an extra task: identify misleading wrong options.

## Why use it

- Strengthens conceptual understanding.
- Surfaces common misconceptions.
- Improves learning depth vs memorization-only answering.

## Example pattern

```md
## Question 1
What does `arr * 3` do for a list?

- Repeats the list three times
- Converts all elements to strings
- Extends the list with three empty elements
- Multiplies each element by 3

Answer: 1
Imposters: 4
Type: single
Time: 30
Explanation: `*` repeats the list, it does not multiply each element.
```

- Correct answer: `1`
- Imposter answer(s): `4`

```bash
quizmd --validate hello-imposter.md
quizmd hello-imposter.md
```
