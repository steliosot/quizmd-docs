# Quickstart

QuizMD is a CLI tool for creating and running interactive quizzes from Markdown files. Use it for teaching, workshops, self-study, or coding classes.

This takes about one minute. Each command block has a copy button.

## 1. Install QuizMD

Install QuizMD from PyPI, then confirm the `quizmd` command is available.

```bash
pip install quizmd
quizmd --version
```

Expected:

```text
quizmd 2.x.x
```

## 2. Create your first quiz

Create ready-to-edit starter templates in your current folder, so you start from working quiz files instead of a blank page.

```bash
quizmd init
```

This creates starter quiz files in your current folder:

```text
Created:
- hello-quiz.md
- hello-imposter.md
- hello-essay.md
```

## 3. Validate the quiz

Validate each starter before you run it. This catches missing fields, broken answer indexes, and essay structure issues early.

<Callout type="warning">
  Always validate your quiz before running it.
</Callout>

```bash
quizmd --validate hello-quiz.md
quizmd --validate hello-imposter.md
quizmd --validate hello-essay.md
```

Expected:

```text
Validation passed: Hello Quiz (3 questions)
Validation passed: Hello Imposter Quiz (3 questions)
Validation passed: Essay Question: requirements basics (3 points)
```

## 4. Run the quiz

Run the classic and imposter starters directly. QuizMD runs files with `quizmd file.md`.

```bash
quizmd hello-quiz.md
quizmd hello-imposter.md
```

You should see an interactive terminal quiz:

```text
Question 1/3
What is 2 + 2?

> 1. 3
  2. 4
  3. 5

Use ↑/↓ to move, Space to select, Enter to submit
```

## 5. Use essay mode

Essay mode requires an AI provider API key. Only essay-based quizzes need this; standard quizzes run without an API key.

Mac/Linux:

```sh
export GEMINI_API_KEY="your_key_here"
quizmd hello-essay.md
```

Windows PowerShell:

```powershell
$env:GEMINI_API_KEY="your_key_here"
quizmd hello-essay.md
```

Windows Command Prompt:

```bat
set GEMINI_API_KEY=your_key_here
quizmd hello-essay.md
```

Supported providers: Gemini, OpenAI, and Anthropic. If you do not have an API key yet, skip essay mode and start with `hello-quiz.md`.

<Callout type="warning">
  **Common mistakes**

  - Forgot `--validate` before running a new file.
  - Forgot an API key before running `hello-essay.md`.
  - Ran `quizmd hello-quiz.md` from a different folder than the starter files.
  - Edited the Markdown file but did not save it before running.
  - Used the wrong Markdown structure (`Answer`, `Type`, sections) in custom files.
</Callout>
