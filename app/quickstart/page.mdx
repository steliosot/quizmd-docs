# Quickstart

## 1) Install

Install QuizMD from PyPI, then check the command is available.

```bash
pip install quizmd
quizmd --version
```

## 2) Init

Create ready-to-edit starter templates in the current folder, so you can begin with working examples instead of a blank file.

```bash
quizmd init
```

## 3) Validate

```bash
quizmd --validate hello-quiz.md
quizmd --validate hello-imposter.md
quizmd --validate hello-essay.md
```

## 4) Run

```bash
quizmd hello-quiz.md
quizmd hello-imposter.md
```

For essay mode set one API key first:

```bash
export GEMINI_API_KEY="your_key_here"
# or OPENAI_API_KEY / ANTHROPIC_API_KEY
quizmd hello-essay.md
```

## What you should see

```text
Validation passed: Hello Quiz (3 questions)
```

Then QuizMD opens an interactive terminal quiz with arrow-key selection, feedback, explanations, and a final score.

<Callout type="warning">
  **Common mistakes**

  - Forgot `--validate` before running a new file.
  - Forgot an API key before running `hello-essay.md`.
  - Used the wrong markdown structure (`Answer`, `Type`, sections) in custom files.
</Callout>
