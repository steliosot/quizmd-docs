# Quickstart

## 1) Install

Install QuizMD from PyPI, then confirm the `quizmd` command is available.

```bash
pip install quizmd
quizmd --version
```

## 2) Init

Create ready-to-edit starter templates in the current folder, so you start from working quiz files instead of a blank page.

```bash
quizmd init
```

## 3) Validate

Validate each starter before you run it. This catches missing fields, broken answer indexes, and essay structure issues early.

```bash
quizmd --validate hello-quiz.md
quizmd --validate hello-imposter.md
quizmd --validate hello-essay.md
```

## 4) Run

Run the classic and imposter starters directly. Essay mode needs one AI provider key first.

```bash
quizmd hello-quiz.md
quizmd hello-imposter.md
```

For essay mode, set one API key first:

```bash
export GEMINI_API_KEY="your_key_here"
# or OPENAI_API_KEY / ANTHROPIC_API_KEY
quizmd hello-essay.md
```

## What you should see

```text
Validation passed: Hello Quiz (3 questions)
```

Then QuizMD opens an interactive terminal quiz with arrow-key selection, instant feedback, explanations, and a final score.

<Callout type="warning">
  **Common mistakes**

  - Forgot `--validate` before running a new file.
  - Forgot an API key before running `hello-essay.md`.
  - Used the wrong Markdown structure (`Answer`, `Type`, sections) in custom files.
</Callout>
