# For Self-Study

## Why QuizMD fits

- Lightweight daily practice.
- Immediate correction loop.
- Easy to create personal quizzes in Markdown.

## Best mode to start with

Classic Mode for recall, then Essay Mode for deeper reflection.

## Suggested workflow

1. Build one small quiz per topic.
2. Validate and run daily.
3. Track weak topics from summary and exports.
