# For Coding Classes

## Why QuizMD fits

- Code blocks render directly in question text.
- Imposter mode catches common coding misconceptions.
- Essay mode supports rubric-based conceptual grading.

## Best modes to start with

Classic Mode + Imposter Mode.

## Best examples to run

- `quizmd quizzes/python-code-blocks-quiz.md`
- `quizmd quizzes/python-basics-imposter.md`

## Suggested workflow

1. Warm-up with classic code-block quiz.
2. Run imposter mode for misconception checks.
3. Use essay prompts for design/explanation tasks.
