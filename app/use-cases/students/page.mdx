# For Students

## Why QuizMD fits

- Fast terminal practice loop.
- Immediate feedback and explanations.
- No heavy setup beyond Python and pip.

## Best mode to start with

Classic Mode, then Imposter Mode for stronger revision.

## Suggested workflow

1. Run starter quizzes from init.
2. Repeat missed topics from the summary.
3. Use imposter quizzes for misconception checks.

```bash
quizmd init
quizmd hello-quiz.md
quizmd hello-imposter.md
```
