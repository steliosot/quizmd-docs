# For Teachers

## Why QuizMD fits

- Fast quiz authoring in plain markdown.
- Repeatable validation before class.
- Multiple modes for formative assessment depth.

## Best mode to start with

Classic Mode.

## Best examples to run

- `quizmd quizzes/general-science-quiz.md`
- `quizmd quizzes/history-and-civics-quiz.md`

## Suggested workflow

1. Start from `quizmd init` template.
2. Customize file to lesson goals.
3. Validate before class.
4. Run and optionally save outputs.
