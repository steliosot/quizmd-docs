# For Teachers

## Why QuizMD fits

- Fast quiz authoring in plain Markdown.
- Repeatable validation before class.
- Multiple modes for different levels of formative assessment.

## Best mode to start with

Classic Mode for quick checks, then Imposter Mode when you want learners to explain misconceptions.

## Best examples to run

```bash
quizmd quizzes/general-science-quiz.md
quizmd quizzes/history-and-civics-quiz.md
```

## Suggested workflow

1. Start from the init template.
2. Customize the file to lesson goals.
3. Validate before class.
4. Run and optionally save outputs.

```bash
quizmd init
quizmd --validate hello-quiz.md
quizmd hello-quiz.md
```
