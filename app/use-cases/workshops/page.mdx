# For Workshops

## Why QuizMD fits

- Shared Markdown files are easy to distribute.
- Works well in constrained environments.
- Clear command flow for onboarding participants quickly.

## Best mode to start with

Classic Mode with starter files, then Room: Compete or Room: Collaborate when the group is ready.

## Suggested workflow

1. Ask everyone to run install and init.
2. Validate together.
3. Run one baseline quiz.
4. Transition to room modes for group activities.
