# For Workshops

## Why QuizMD fits

- Shared markdown files are easy to distribute.
- Works well in constrained environments.
- Clear command flow for onboarding participants.

## Best mode to start with

Classic Mode with starter files.

## Suggested workflow

1. Ask everyone to run install + init.
2. Validate together.
3. Run one baseline quiz.
4. Transition to room modes for group activities.
