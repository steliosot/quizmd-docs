# Beginner Examples

## Harry Potter Quiz

- Demonstrates: approachable trivia in the Classic quiz format.
- Difficulty: easy to medium.
- Mode: Classic.
- Learn from it: question pacing and friendly explanations.

```bash
quizmd quizzes/harry-potter-quiz.md
```

```md
## Question 1
Who is Harry Potter's best friend?
```
