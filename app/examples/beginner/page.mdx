# Beginner Examples

## Harry Potter Quiz

- Demonstrates: approachable general trivia format.
- Difficulty: easy to medium.
- Mode: Classic.
- Learn from it: question pacing, explanation style.

```bash
quizmd quizzes/harry-potter-quiz.md
```

```md
## Question 1
Who is Harry Potter's best friend?
```
